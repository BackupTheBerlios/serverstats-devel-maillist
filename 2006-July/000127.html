Hello,

I believe that I found a really good mean to use variables and 
parameters from xml
In Xforms we call this technic  "Attribute Value Template" (AVT) that 
consists to  mix xpath expression between "{ }"  chars and normal text

<graph name="snmp1" >
<param name="title" value="Load {../@name}" />
</graph>

is the same as

<graph name="snmp1" >
<param name="title">Load snmp1</param>
</graph>



This demo use avt to create the source name

http://ckr.zeninteractif.com/serverstats-svn/detail.php?graph=0&tree=&filter=

Look at the latest revision of xmlconfig2 branch  for source
